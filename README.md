# 📄 PDF 图片一键提取助手 (PDF Image Extractor)

这是一个简单、高效的工具，专门用于从 PDF 文档中提取所有嵌入的图片。无需安装复杂的图片处理软件，一键即可搞定。

---

### ✨ 主要功能

* **全自动提取**：自动识别并导出 PDF 页面中的所有内嵌图片。
* **智能分类**：为每个 PDF 文件自动创建一个同名文件夹。
* **规范命名**：图片按 `PDF文件名_页码_图片序号` 规则自动编号。
* **零配置使用**：双击即可运行，支持路径直接粘贴或文件拖入。

---

### 🚀 如何使用

1. **启动程序**：双击运行 `PDF图片提取器.exe`。
2. **输入路径**：
* 你可以手动输入 PDF 文件的完整路径。
* **更简单的办法**：直接将 PDF 文件用鼠标**拖进**弹出的黑色窗口中。


3. **按下回车**：程序会自动开始工作。
4. **查看结果**：提取完成后，在 `.exe` 同级目录下会生成一个与 PDF 同名的文件夹，里面就是你要的图片。

---

### 📂 输出示例

假设你的文件名为 `工作报告.pdf`，提取后的结构如下：

```text
/工具所在目录
    📄 PDF图片提取器.exe
    📂 工作报告                <-- 自动生成的文件夹
        📷 工作报告_1_1.png     (第1页第1张图)
        📷 工作报告_1_2.jpg     (第1页第2张图)
        📷 工作报告_2_1.png     (第2页第1张图)

```

---

### ⚠️ 注意事项

* **扫描件说明**：本工具提取的是 PDF 中**嵌入**的图片对象。如果你的 PDF 是整页扫面件且没有其他插图，提取出的可能是整页的图片。
* **杀毒软件**：由于是个人编写的未签名工具，Windows Defender 或杀毒软件可能会弹出警告，请选择“仍要运行”或添加信任。
* **运行环境**：本程序已完成打包，无需安装 Python 即可在 Windows 系统直接运行。
